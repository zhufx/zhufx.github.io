Hugo教程

Mar 24, 2020 · 1 min read

Hugo命令

最近想要写一些技术博客，看了一些教程，最后选择hugo搭建静态的博客网站。

选择hugo有几个原因吧：

  1. 简单易搭建
  2. 好看的主题
  3. 速度快

经过一些踩坑和试错一次后，成功搭建了自己的静态网站，把一些常用的命令记录下来吧。

hugo命令

hugo new post/new.md

这条命令用来新建自己的文章，可以自动生成一个markdown文件，并且自动生成文章

hugo server -D -t soho

这条命令用来编译自己的本地静态文件，-D代表编译草稿状态下的稿件，-t用来选择主题，默认地址是loacalhost:1313

hugo --baseUrl https://zhufx.github.io/ -D -t soho

这条命令是用来编译本地静态文件到public文件夹下，通过提交public文件夹下的静态网页内容到github便可以更新到自己的博客内容，提交命令如下：

git add -A
git commit -m "message"
git push -u origin maste

这里只是做一个简单的记录，其实不够详细，后面有时间再探究下hugo的高阶用法吧。

遇到的坑

遇到的一个主要问题是把public目录下的静态文件提交到github后css加载不了，主要问题是github现在已经不能访问http的资源了。自2016年6月15日起，使用Github Pages默认域名搭建个人站点，已被强制要求HTTPS访问，即使用HTTP访问，也会被重定向到HTTPS。

所以需要将baseUrl改成https即可，当然也可以在config.toml中设置。
